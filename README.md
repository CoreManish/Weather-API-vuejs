### Weather-Manager
#### Technology used
- HTML
- CSS
- Bootstrap
- VueJS

#### What is
You can manage weather of multiple time zone at a time right in the web browser

#### How to run
- Place your own openweather API key in line number 24 and 96 inside vuejs.js
- Open index.html in the browser
